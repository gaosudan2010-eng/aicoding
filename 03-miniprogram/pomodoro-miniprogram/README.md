# 🍅 番茄钟小程序

一款专注工作、高效休息的番茄钟计时器小程序，采用 iOS 设计规范。

## ✨ 功能特性

### 核心功能
- ⏱️ **番茄计时** - 25分钟工作 + 5分钟休息的经典番茄工作法
- 📊 **今日统计** - 实时显示完成番茄数、专注分钟、连续番茄
- ⚙️ **自定义设置** - 可调整工作时长、休息时长、长休息间隔
- 💾 **数据持久化** - 自动保存进度和设置

### 提醒功能
- 📳 **震动提醒** - 计时结束时震动通知
- 🔔 **声音提醒** - 可选的声音提示

### 设计特点
- 🎨 **iOS 风格** - 遵循 iOS Human Interface Guidelines
- 🍅 **可爱番茄** - 纯 CSS 绘制的番茄图形
- ✨ **进度动画** - 流畅的进度环动画和光点效果
- 🌈 **主题切换** - 工作/休息模式自动切换背景色

## 🚀 开发指南

### 1. 导入项目
1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择本项目目录
4. 填入你的 AppID（或使用测试号）

### 2. 开发调试
- 使用模拟器预览效果
- 真机调试测试震动和声音

### 3. 发布上线
1. 在开发者工具中点击「上传」
2. 登录微信公众平台提交审核
3. 审核通过后发布

## 🎯 使用说明

1. **开始专注** - 点击「开始专注」按钮开始 25 分钟计时
2. **暂停/继续** - 随时暂停或继续计时
3. **跳过阶段** - 点击「⏭」跳过当前阶段
4. **重置计时** - 点击「↺」重置当前阶段
5. **设置** - 点击右上角齿轮图标调整参数
