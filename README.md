## 🎯 学习路线

### 第一阶段：认识 AI Coding（1-2天）
- 理解 AI Coding 新范式
- 熟悉 Cursor 界面和快捷键
- 完成第一个 Hello World

### 第二阶段：代码基础认知（3-5天）
- HTML 基础：网页的骨架
- CSS 基础：网页的皮肤
- JavaScript 基础：网页的大脑

### 第三阶段：实战项目（1-2周）
- 计数器 - 学习 JS 交互
- Todo List - 学习数据存储
- 个人作品集 - 综合练习

### 第四阶段：高效 AI 协作（持续）
- Prompt 技巧
- 调试方法
- 迭代优化

### 第五阶段：进阶学习（2-4周）
- React/Vue 框架
- 微信小程序
- App 开发

## 🛠️ 必会快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd + L` | 打开 AI 对话 |
| `Cmd + K` | 代码中呼出 AI |
| `Cmd + I` | 打开 Composer |
| `Cmd + S` | 保存文件 |
| `Cmd + Z` | 撤销 |

## 💡 Prompt 万能模板

```
【背景】我是产品经理，正在做 XX 项目
【目标】需要实现 XX 功能
【具体要求】
1. 功能点 1
2. 功能点 2
3. 设计要求：简洁现代/科技感/小红书风格...
【技术限制】只使用 HTML/CSS/JS，不需要框架
```

## 🎉 产品经理的优势

你天天写 PRD，这就是最好的 Prompt 训练！需求文档写得越清晰，AI 生成的代码越准确。
